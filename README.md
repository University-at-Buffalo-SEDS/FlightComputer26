# Flight Computer 26

This is the Flight Computer repository for UB SEDS 2026 IREC.

## Communication

[OpenProject board](https://projects.rylanswebsite.com/projects/avionics-2025)

## Building
Build scripts are provided for both debug and release builds, scripts are also provided to both build and flash the firmware to the mcu. to run make sure python is installed and then run the associated build script.

When building, if you want to exclude the telemetry library build pass no-telemetry to the build script
